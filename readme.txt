JUnit Testing of Trig Functions.

Implementation of a set of functions (sin, tan, cos, cot, etc.) using Java and GitHub for project management.
JUnit 4.0 used for creating detailed tests to ensure accuracy of functions and invalid usage exceptions.

A demonstration is available through the video "Video.mp4" or on YouTube (https://www.youtube.com/watch?v=Ww__5QXWfmw).
